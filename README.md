# Coding Quiz App

## Description
An interactive coding quiz application built using HTML, CSS, and JavaScript. It includes timer-based questions, skip functionality, detailed reports, and performance tracking.

## Features
- Multiple-choice questions (HTML, CSS, JavaScript)
- Timer for each question (15 seconds per question)
- Skip question feature
- Detailed report with correct, wrong, and skipped answers
- Chart visualization of performance
- Dark-themed modern UI
- Responsive design
- Performance tracking with explanations

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## Features Details

### Quiz Functionality
- **18 Questions**: Covering HTML, CSS, and JavaScript topics
- **Timer System**: 15-second countdown per question with visual warnings
- **Skip Option**: Users can skip questions without penalty
- **Answer Validation**: Immediate feedback on answer selection
- **Progress Tracking**: Visual progress bar and question counter

### Report System
- **Comprehensive Reports**: Shows all questions with user answers and correct answers
- **Performance Stats**: Counts of correct, wrong, and skipped questions
- **Chart Visualization**: Animated bar chart showing performance distribution
- **Color Coding**: 
  - Green: Correct answers
  - Red: Wrong answers
  - Yellow: Skipped questions

### User Interface
- **Dark Theme**: Professional dark gradient background
- **Glassmorphism**: Modern glass-like UI elements
- **Responsive Design**: Works on desktop and mobile devices
- **Smooth Animations**: Transitions and hover effects
- **Confetti Celebration**: Animated confetti for high scores
